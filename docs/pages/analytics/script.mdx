# Adding the Insights Tracker in Multiple Languages

The Insights tracker can be integrated into various frameworks and languages. Below are examples for different platforms:

```javascript filename="_document.js"
import { Html, Head, Main } from 'next/document';
import Script from 'next/script';

export default function Document() {
  return (
    <Html>
      <Head>
        <Script
          src="https://collector.insihts.com/script.js"
          strategy="afterInteractive"
          data-website-id="YOUR_WEBSITE_ID"
          onLoad={() => console.log('Insights script loaded')}
          onError={(error) => console.error('Insights script failed to load:', error)}
        />
      </Head>
      <body>
        <Main />
      </body>
    </Html>
  );
}
```

## Accessing the Tracking Code in Insihts

Once you add a website in Insihts, you can obtain the tracking code using the following steps:

1. Navigate to the **Highlights** page in Insihts.
   - Click the **Get Tracking Code** button.

2. Alternatively, click **View Details** for the website.
   - Under the **Info** section, navigate to the **Tracking Code** tab.
   - Copy the tracking code provided.

3. Another option is to go to the **Products** section.
   - Click the **View CTA** button, which redirects you to the **Overview** page.
   - From there, navigate to the **Info** page and select the **Tracking Code** tab to find the tracking code.
